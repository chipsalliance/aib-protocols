// Data and Channel Size
  Logic Link Data Info
    LogicLink MAIN     downstream TX 1067  RX    0
    LogicLink MAIN     upstream TX    0  RX 1067
                                -------  -------
    Total     MAIN              TX 1067  RX 1067

  Channel Info
    MAIN: Each channel is Gen2 PHY running at Quarter Rate with 320 bits
    MAIN: 4x channels
    MAIN: Total AIB bits is 1280 bits

       TX: No DBI
       TX: Persistent Strobe adds 1 overhead bits per channel
       TX: Persistent Marker adds 4 overhead bits per channel
       TX: Total overhead bits across 4 channels is 20
       TX: Total data bits available 1260

       RX: No DBI
       RX: Persistent Strobe adds 1 overhead bits per channel
       RX: Persistent Marker adds 4 overhead bits per channel
       RX: Total overhead bits across 4 channels is 20
       RX: Total data bits available 1260

  MAIN TX needs 1067 bits of data and has 1260 bits available across 4x Gen2 Quarter Rate channels so  193 spare bits
  MAIN RX needs 1067 bits of data and has 1260 bits available across 4x Gen2 Quarter Rate channels so  193 spare bits
