//////////////////////////////////////////////////////////////////////
// Data and Channel Size
  Logic Link Data Info
    LogicLink MAIN     downstream TX 1067  RX    0
    LogicLink MAIN     upstream TX    0  RX 1067
                                -------  -------
    Total     MAIN              TX 1067  RX 1067

  Channel Info
    MAIN: Each channel is Gen1Only PHY running at Half Rate with 80 bits
    MAIN: 16x channels
    MAIN: Total AIB bits is 1280 bits

       TX: No DBI
       TX: Persistent Strobe adds 1 overhead bits per channel
       TX: Persistent Marker adds 2 overhead bits per channel
       TX: Total overhead bits across 16 channels is 48
       TX: Total data bits available 1232

       RX: No DBI
       RX: Persistent Strobe adds 1 overhead bits per channel
       RX: Persistent Marker adds 2 overhead bits per channel
       RX: Total overhead bits across 16 channels is 48
       RX: Total data bits available 1232

  MAIN TX needs 1067 bits of data and has 1232 bits available across 16x Gen1Only Half Rate channels so  165 spare bits
  INFORMATION: At least one full channel unused for TX
  MAIN RX needs 1067 bits of data and has 1232 bits available across 16x Gen1Only Half Rate channels so  165 spare bits
  INFORMATION: At least one full channel unused for RX

// Data and Channel Size
