// Data and Channel Size
  Logic Link Data Info
    LogicLink MAIN     downstream TX  269  RX    0
    LogicLink MAIN     upstream TX    0  RX  269
                                -------  -------
    Total     MAIN              TX  269  RX  269

  Channel Info
    MAIN: Each channel is Gen2 PHY running at Quarter Rate with 320 bits
    MAIN: 1x channels
    MAIN: Total AIB bits is 320 bits

       TX: No DBI
       TX: Persistent Strobe adds 1 overhead bits per channel
       TX: Persistent Marker adds 4 overhead bits per channel
       TX: Total overhead bits across 1 channels is 5
       TX: Total data bits available 315

       RX: No DBI
       RX: Persistent Strobe adds 1 overhead bits per channel
       RX: Persistent Marker adds 4 overhead bits per channel
       RX: Total overhead bits across 1 channels is 5
       RX: Total data bits available 315

  MAIN TX needs  269 bits of data and has  315 bits available across 1x Gen2 Quarter Rate channels so   46 spare bits
  MAIN RX needs  269 bits of data and has  315 bits available across 1x Gen2 Quarter Rate channels so   46 spare bits
