// Data and Channel Size
  Logic Link Data Info
    LogicLink MAIN     downstream TX  145  RX    0
    LogicLink MAIN     upstream TX    0  RX  145
                                -------  -------
    Total     MAIN              TX  145  RX  145

  Channel Info
    MAIN: Each channel is Gen2 PHY running at Full Rate with 80 bits
    MAIN: 2x channels
    MAIN: Total AIB bits is 160 bits

       TX: No DBI
       TX: Persistent Strobe adds 1 overhead bits per channel
       TX: Persistent Marker adds 1 overhead bits per channel
       TX: Total overhead bits across 2 channels is 4
       TX: Total data bits available 156

       RX: No DBI
       RX: Persistent Strobe adds 1 overhead bits per channel
       RX: Persistent Marker adds 1 overhead bits per channel
       RX: Total overhead bits across 2 channels is 4
       RX: Total data bits available 156

  MAIN TX needs  145 bits of data and has  156 bits available across 2x Gen2 Full Rate channels so   11 spare bits
  MAIN RX needs  145 bits of data and has  156 bits available across 2x Gen2 Full Rate channels so   11 spare bits
