// Data and Channel Size
  Logic Link Data Info
    LogicLink MAIN     downstream TX  141  RX    0
    LogicLink MAIN     upstream TX    0  RX  141
                                -------  -------
    Total     MAIN              TX  141  RX  141

  Channel Info
    MAIN: Each channel is Gen2 PHY running at Half Rate with 160 bits
    MAIN: 1x channels
    MAIN: Total AIB bits is 160 bits

       TX: No DBI
       TX: Persistent Strobe adds 1 overhead bits per channel
       TX: Persistent Marker adds 2 overhead bits per channel
       TX: Total overhead bits across 1 channels is 3
       TX: Total data bits available 157

       RX: No DBI
       RX: Persistent Strobe adds 1 overhead bits per channel
       RX: Persistent Marker adds 2 overhead bits per channel
       RX: Total overhead bits across 1 channels is 3
       RX: Total data bits available 157

  MAIN TX needs  141 bits of data and has  157 bits available across 1x Gen2 Half Rate channels so   16 spare bits
  MAIN RX needs  141 bits of data and has  157 bits available across 1x Gen2 Half Rate channels so   16 spare bits
