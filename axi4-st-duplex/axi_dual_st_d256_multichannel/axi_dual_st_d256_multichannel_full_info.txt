// Data and Channel Size
  Logic Link Data Info
    LogicLink RSTRUCT  ST_M2S   TX  257  RX    1
    LogicLink RSTRUCT  ST_S2M   TX    1  RX  257
    RepStruct in Full Mode           x1       x1
                                -------  -------
    Total     RSTRUCT           TX  258  RX  258

  RSTRUCT Sub Channel Info
  Note: RSTRUCT describes the Replicated Struct on a Full rate channel.
        RSTRUCT will be replicated for Full rate per configuration and that is known as MAIN channel

    RSTRUCT: Each channel is Gen1Only PHY running at Full Rate with 40 bits
    RSTRUCT: 7x channels
    RSTRUCT: Total AIB bits is 280 bits

  MAIN Channel Info
    MAIN: Each channel is Gen1Only PHY running at Full Rate with 40 bits
    MAIN: 7x channels
    MAIN: Total AIB bits is 280 bits

       TX: No DBI
       TX: Strobe is Recoverable or non-existent
       TX: Marker is Recoverable or non-existent
       TX: Total RSTRUCT overhead bits across 7 Full Rate channels is 0
       TX: Total RSTRUCT data bits available 280
       TX: Total MAIN overhead bits across 7 Full channels is 0
       TX: Total MAIN data bits available 280

       RX: No DBI
       RX: Strobe is Recoverable or non-existent
       RX: Marker is Recoverable or non-existent
       RX: Total RSTRUCT overhead bits across 7 Full Rate channels is 0
       RX: Total RSTRUCT data bits available 280
       RX: Total MAIN overhead bits across 7 Full channels is 0
       RX: Total MAIN data bits available 280

  RSTRUCT TX needs  258 bits of data and has  280 bits available across 7x Gen1Only Full Rate channels so   22 spare bits
  RSTRUCT RX needs  258 bits of data and has  280 bits available across 7x Gen1Only Full Rate channels so   22 spare bits

  MAIN TX needs  258 bits of data and has  280 bits available across 7x Gen1Only Full Rate channels so   22 spare bits
  MAIN RX needs  258 bits of data and has  280 bits available across 7x Gen1Only Full Rate channels so   22 spare bits
